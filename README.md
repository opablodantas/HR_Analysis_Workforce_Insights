
# Análise de Dados da Força de Trabalho

Bem-vindo ao projeto de **Análise de Dados da Força de Trabalho**. Este repositório contém um conjunto de dados e análises relacionadas aos funcionários de uma empresa, incluindo informações sobre educação, experiência, demografia e fatores de emprego. O objetivo deste projeto é extrair insights valiosos para aprimorar a gestão de recursos humanos e a tomada de decisões estratégicas.

## Descrição do Projeto

Este projeto se concentra na análise de dados da força de trabalho de uma empresa e está dividido em **duas partes principais**:

1. **Relatório Analítico no Jupyter Notebook**

   * Nesta primeira etapa, exploramos os dados e realizamos análises detalhadas sobre a distribuição de qualificações educacionais, variação no tempo de serviço com base na cidade, correlação entre nível de pagamento e experiência, distribuição de gênero, padrões de licença e outros insights relevantes.
   * Este notebook já está praticamente pronto e fornece uma base sólida para a compreensão do comportamento da força de trabalho.

2. **Deploy do Modelo Preditivo**

   * Na segunda etapa, será desenvolvido um **modelo preditivo** que ajuda a informar se um funcionário sair da empresa.
   * Será criada uma **aplicação interativa**, onde o usuário poderá inserir informações de um colaborador e o sistema retornará uma previsão de permanência ou saída da empresa.
